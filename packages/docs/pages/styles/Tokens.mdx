# Spacing

These are spacing tokens. Use this tokens to maintain your semantic variables consistent with the general spacing.

```
:root {
  --space-4: @space-4; /* 4px */
  --space-8: @space-8; /* 8px */
  --space-12: @space-12; /* 12px */
  --space-16: @space-16; /* 16px */
  --space-24: @space-24; /* 24px */
  --space-32: @space-32; /* 32px */
  --space-40: @space-40; /* 40px */
  --space-48: @space-48; /* 48px */
  --spacer: var(--space-8); /* 8px */
}
```
